# vocollect_tools
## Description
Tools for analyzing and visualizing vocollect log files.

Language: c++

Dependencies: sfml

## Installation
### Install prerequisites
* gcc
* sfml

### Build
```
$ git clone https://github.com/DennisLovkvist/vocollect_tools.git
$ cd vocollect_tools

$ ./vcclean/build.ps1
$ ./vcgenpick/build.ps1
$ ./vcnetstat/build.ps1
$ ./vcnetmap/build.ps1

$ ./install.sh
```
